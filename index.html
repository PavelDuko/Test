Список команд в "git"

git clone - Клонирование удаленного репозитория

git fetch — скачивает изменения, но не сливает их с вашей текущей веткой

git pull — скачивает изменения, и автоматом сливает их с текущей веткой

git status — показывает статус изменения файлов

git add <имя_файла> добавляет файл в индекс
git add добавляетвсе изменения в текущем каталоге
git commit - m "Описание" Создает комит с описанием изменений git log росмотр истории комитов git checkout -- <имя_файла> Отмена последних изменений git branch <имя> создание новой ветки git checkout <имя_ветки> переключение на нужную ветку git checkout -b <имя_ветки> Создание и переход на новую ветку git merge <имя_ветки> слияние изменений из указанной ветки в текущую git branch -d <имя_ветки> удаляет указанную ветку git checkout -b <имя_ветки> Создание и переход на новую ветку
